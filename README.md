# Basic Pong

A simple Pong game built with Python and pygame.

## About
This is a basic implementation of the classic Pong game.  
Built to practice game loops, collision detection, and real time input handling.

## Gameplay
- Two player paddle controls
- Ball bounces off walls and paddles
- Ball resets when a point is scored

## Controls
- Left paddle: W / S  
- Right paddle: Up Arrow / Down Arrow  

## Tech
- Python 3
- pygame

## How to Run
1. Install pygame
2. Run the game

## Why This Exists
A learning project focused on understanding core game mechanics and pygame basics.

## Notes
Clean, minimal code that can be expanded with scoring, sound, or AI opponents.
